# allq_client

Crystal-lang support for Allqueue.

Beta: In progress. 

## Installation

1. Add the dependency to your `shard.yml`:
```yaml
dependencies:
  allq_client:
    github: blitline_dev/allq_client_shard
```
2. Run `shards install`

## Usage

```crystal
require "allq_client"
```
